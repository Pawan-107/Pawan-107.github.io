## Objective

To understand different types of function arguments, variable scope, recursive functions, and list data structures along with their common operations.

## Topics Covered

### 1. Types of Function Arguments

#### Positional Arguments

Arguments are passed according to their position in the function definition.

#### Default Arguments

Parameters are assigned default values that are used when no argument is provided.

#### Keyword Arguments

Arguments are passed using parameter names and can be supplied in any order.

### 2. Variable-Length Arguments

#### *args

Used to pass a variable number of non-keyword arguments to a function.

Applications:

* Expense calculations
* Processing multiple numeric inputs

#### **kwargs

Used to pass a variable number of keyword arguments to a function.

Applications:

* Model hyperparameter configuration
* Flexible function inputs

### 3. Variable Scope

#### Local Variables

Variables declared inside a function and accessible only within that function.

#### Global Variables

Variables declared outside functions and accessible throughout the program.

Concepts Learned:

* Accessing global variables
* Modifying global variables using the `global` keyword

### 4. Recursive Functions

A recursive function calls itself until a base condition is reached.

Components:

* Base Case
* Recursive Case

Programs Implemented:

* Factorial calculation using recursion
* Fibonacci series generation using recursion

### 5. Data Structures

#### Lists

A list is an ordered and mutable collection used to store multiple items.

### 6. List Operations

#### Append and Extend

* `append()` adds a single element.
* `extend()` adds multiple elements from another iterable.

#### Insert and Remove

* `insert(position, item)`
* `remove(item)`
* `pop()`
* `pop(index)`

#### List Slicing

Accessing specific portions of a list using indexing and slicing techniques.

#### Sorting

* `sort()`
* `sorted()`
* Reverse sorting
* Finding maximum values

### 7. Practical List Programs

#### Salary Processing

Applied percentage-based deductions to salary values stored in a list.

#### Pair Sum Problem

Identified unique pairs of numbers whose sum equals 11.

#### Outlier Removal

Removed invalid values (`-99.0`) from a dataset using loops and list operations.
