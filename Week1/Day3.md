# Day 3 - While Loops and Functions in Python

**Date:** 24/06/2026

## Objective

To understand the implementation of real-world applications using while loops and to learn the concept of functions, including built-in functions, user-defined functions, parameters, arguments, and return statements.

## Topics Covered

### 1. While Loop Applications

#### ATM Simulator

Developed a menu-driven ATM simulator using a while loop.

Features:

* Check account balance
* Deposit money
* Withdraw money
* Exit the application

Concepts Used:

* Infinite loops using `while True`
* Conditional statements
* User input handling
* Loop termination using `break`

#### Shopping Cart System

Created a simple shopping cart application.

Features:

* Add items to cart
* View total items and cost
* Checkout and payment confirmation

Concepts Used:

* Variables for maintaining totals
* Repetitive user interaction
* Conditional branching

### 2. Built-in Functions

Studied commonly used Python built-in functions:

* `len()` – Returns the length of a sequence
* `sum()` – Calculates the sum of elements
* `max()` – Returns the maximum value
* `min()` – Returns the minimum value
* `abs()` – Returns the absolute value
* `round()` – Rounds a number to specified decimal places
* `print()` – Displays output

### 3. User-Defined Functions

Learned how to create and use functions using the `def` keyword.

Example Activities:

* Welcome message function
* Sum of two numbers
* Arithmetic operation function

Benefits:

* Code reusability
* Improved readability
* Easier maintenance

### 4. Parameters and Arguments

Learned the difference between:

* **Parameters** – Variables defined in a function declaration.
* **Arguments** – Values passed when calling a function.

Implemented functions that accepted multiple parameters and performed arithmetic operations.

### 5. Return Statement

Learned how functions return values using the `return` keyword.

Example:

* Square of a number function
* Returning calculated values to the calling program

### 6. Prime Number Check Using Functions

Implemented a function to determine whether a number is prime.

Concepts Used:

* Loops
* Conditional statements
* Function return values
