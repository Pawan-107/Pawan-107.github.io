## Topics Covered

### 1. Tuples

A tuple is an ordered and immutable collection of elements.

Features:

* Ordered collection
* Immutable in nature
* Allows duplicate values
* Faster than lists for fixed data

### 2. Tuple Unpacking

Learned how to extract tuple values into individual variables.

#### Basic Unpacking

Assigned tuple elements directly to variables.

#### Wildcard Unpacking

Used `*` to collect multiple values into a list during unpacking.

Applications:

* Storing student records
* Handling grouped datasets

### 3. Working with Datasets Using Tuples

* Converted tuple to list
* Performed modifications

### 5. Dictionaries

A dictionary stores data as key-value pairs.

Features:

* Fast data retrieval
* Unique keys
* Mutable structure

Methods Studied:

* `keys()`
* `values()`
* `items()`

Operations Performed:

* Accessing dictionary data
* Updating values
* Adding new key-value pairs
* Iterating through dictionary items

### 6. Counting Occurrences Using Dictionary

Applications:

* Data analysis

* Frequency distribution

### 7. Sets

A set is an unordered collection of unique elements.

Features:

* Does not allow duplicate values
* Useful for mathematical set operations
* Fast membership checking

### 8. Set Operations

#### Union

Combines all unique elements from multiple sets.

#### Intersection

Returns common elements between sets.

#### Difference

Returns elements present in one set but not in another.
