# Day 2 - Operators, Conditional Statements, and Loops

**Date:** 23 June 2026

## Objective

To learn advanced Python operators, conditional branching, loops, and loop control statements for solving programming problems efficiently.

## Topics Covered

### 1. Bitwise Operators

Studied bit-level operations in Python.

* Bitwise AND (`&`)
* Bitwise OR (`|`)
* Bitwise XOR (`^`)
* Bitwise NOT (`~`)
* In a Python Program, Formula: `~N = -(N + 1)`
* Left Shift (`<<`)
* Right Shift (`>>`)

Also learned:

* Sign bit representation
* 1's Complement
* 2's Complement

### 2. Membership Operators

Used membership operators to check the presence of elements in collections.

* `in`
* `not in`

### 3. Identity Operators

Learned the difference between object identity and value equality.

* `is`
* `is not`
* Difference between `is` and `==`

### 4. Conditional Statements

Implemented decision-making using:

* `if`
* `elif`
* `else`

Programs performed:

* Check whether a number is strictly positive and even.
* Check whether a number is divisible by 3 or 5.
* Age classification program (Child, Teenager, Adult, Senior Citizen).

### 5. For Loop

Learned iteration using:

```python
for i in range(start, stop, step):
```

Activities:

* Printed multiplication tables.
* Iterated through lists.
* Displayed programming languages and their lengths.
* Checked password strength using string length.

### 6. While Loop

Used loops that execute repeatedly while a condition remains true.

Example:

* Counter program using a while loop.

### 7. Loop Control Statements

#### Break Statement

* Terminates the loop immediately when a condition is met.

#### Continue Statement

* Skips the current iteration and moves to the next iteration.

#### Pass Statement

* Placeholder statement used when no action is required.

### 8. For-Else Loop

Learned how the `else` block executes when a loop completes normally without encountering a `break` statement.

Example:

* Searching for a fruit in a list.

### 9. Password Validation Program

Created a password verification system that:

* Allows three attempts.
* Grants access when the correct password is entered.
* Displays a timeout message after three failed attempts.
