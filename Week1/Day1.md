# Day 1 - Python Basics and Operators

**Date:** 22 June 2026

## Objective
To understand the fundamentals of Python programming, including variables, data types, mutability, type conversion, operators, and user input.

## Topics Covered

### 1. Variables and Objects
- Learned about variables and object references.
- Used `type()` to determine the data type of an object.
- Used `id()` to view the memory address of an object.
- Studied reference sharing between variables.

### 2. Dynamic Typing
- Python automatically determines the data type of a variable based on the assigned value.
- No explicit type declaration is required.

### 3. Data Types

#### Numeric Types
- Integer (`int`)
- Float (`float`)
- Complex (`complex`)

#### String Types
- Single-line strings
- Multi-line strings

#### Boolean and None Types
- `True`
- `False`
- `None`

### 4. Mutability and Immutability
- Discussed mutable and immutable objects.
- Understood how object values can or cannot be modified after creation.

### 5. Type Conversion
- Converted values between different data types using functions such as:
  - `int()`
  - `float()`
  - `str()`

### 6. Operators

#### Arithmetic Operators
- `+` Addition
- `-` Subtraction
- `*` Multiplication
- `/` Division
- `//` Floor Division
- `%` Modulus
- `**` Exponentiation

#### Comparison Operators
- `==` Equal to
- `!=` Not equal to
- `>` Greater than
- `<` Less than
- `>=` Greater than or equal to
- `<=` Less than or equal to

#### Logical Operators
- `and`
- `or`
- `not`

#### Assignment Operators
- `=`
- `+=`
- `-=`
- `*=`
- `/=`
- `//=`
- `%=`
- `**=`

#### Bitwise Operators
- `&` (AND)
- `|` (OR)
- `^` (XOR)
- `~` (NOT)
- `<<` (Left Shift)
- `>>` (Right Shift)
